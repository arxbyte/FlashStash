# FlashStash 🚀
A micro-service for instant file and text exchange on a local network.

![FlashStash Interface](screenshots/main_screen.png)

## 📦 How to download and launch the portable version:
1. Download the `FlashStash_Portable` archive.
2. Unzip the archive into any convenient folder (preferably to your desktop).
3. Run **Run.bat**.
4. The program will start a local server. Check the console window and look for the line starting with `Running on http://...` (e.g., `Running on http://xxx.xxx.x.xx:xxxx`).
5. Open this address in a browser on your PC or any device connected to the same Wi-Fi network.

## 🛠 Key Features:
* **Text Exchange:** Instantly share notes, clipboard content, or links between devices.
* **File Transfer:** Upload and download files of any format with no size limits.
* **QR Code:** Quick connection from smartphones or tablets by scanning the on-screen QR code.
* **Password Protection:** Option to restrict access to your stash from other users on the network.

## 📂 Repository Structure:
* `main.py` — the source code of the application.
* `Run.bat` — startup script for the portable Python environment.
