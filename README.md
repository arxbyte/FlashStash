# Flash Stash Portable 🚀
A micro-service for instant file and text exchange on a local network.

## How to download and launch the finished version:
1. Download the 'FlashStash_Portable` archive.
2. Unzip the archive to any convenient folder (preferably on your desktop).
3. Run the **Run.bat**.
4. The program will open the service in the browser!

## Repository structure:
main.py - the source code of the application.
Run.bat is a startup script for the portable Python environment.
